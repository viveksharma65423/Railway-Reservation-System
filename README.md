# Railway Reservation System

A console-based Railway Reservation System developed using Core Python. This project demonstrates Object-Oriented Programming (OOP), File Handling, Exception Handling, and Menu-Driven Programming concepts.

## Features

- User Signup & Login
- View Available Trains
- Book Train Tickets
- Generate Unique PNR Number
- View Booked Tickets
- Cancel Tickets
- Automatic Seat Management
- Data Persistence using Text Files
- Input Validation & Exception Handling

## Technologies Used

- Python 3
- Core Python
- Object-Oriented Programming (OOP)
- File Handling
- Exception Handling

## Files

- `railway.py` – Main source code
- `users.txt` – Stores user credentials
- `tickets.txt` – Stores booked ticket details
